Bubble Sort

Bubble Sort: It is a simple algorithm which is used to sort a given set of n elements provided in form of an array with n number of elements.Sorting takes place by stepping through all the elements one-by-one and comparing it with the adjacent element and swapping them if required.

Time Complexity

Worst and Average Case Time Complexity : O(n*n). Worst case occurs when array is reverse sorted.
Best Case Time Complexity : O(n). Best case occurs when array is already sorted.
